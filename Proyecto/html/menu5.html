Destructuring arrays

var arr = [ 1 , 2 , 3 , 4 ]; var un = arr [ 0 ]; var b = arr [ 1 ]; var c = arr [ 2 ]; var d = arr [ 3 ]; dejar que [a, b, c, d] = [ 1 , 2 , 3 , 4 ]; consola . log (a); // 1 consola . log (b); // 2

Descripcion

Desestructuración nos permite extraer los valores de las matrices y los objetos (incluso muy anidadas) y almacenarlos en las variables con una sintaxis más conveniente.
