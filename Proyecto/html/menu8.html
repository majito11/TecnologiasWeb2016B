Default parameters, rest parameters, named parameters

Parametros por defecto function addTwoNumbers(x, y) { x = x || 0; y = y || 0; return x + y; } En ES6, simplemente podemos suministrar valores por defecto para los parámetros en una función: Parametros de descanso function logArguments() { for (var i=0; i < arguments.length; i++) { console.log(arguments[i]); } } Parametros con nombre function initializeCanvas(options) { var height = options.height || 600; var width = options.width || 400; var lineStroke = options.lineStroke || 'black'; }

Descripcion

En ES5, que habíamos variando las formas de manejar funciones que necesitan los valores por defecto , argumentos indefinidos , y los parámetros con nombre . Con ES6, podemos lograr todo esto y mucho más utilizando una sintaxis más concisa.
