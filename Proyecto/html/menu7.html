Modules export and import

Exporting in ES6 La exportación en CommonJS módulo . exportaciones = 1 ; módulo . exportaciones = {ejem : ' bar ' }; módulo . exportaciones = [ ' foo ' , ' bar ' ]; módulo . exportaciones = función de barra () {}; La exportacion en ES6 export let name = 'David'; export let age = 25; Asi como la exportacion de objetos function sumTwo(a, b) { return a + b; } function sumThree(a, b, c) { return a + b + c; } export { sumTwo, sumThree };

Descripcion

Antes de la ES6, utilizamos las bibliotecas como Browserify para crear módulos en el lado del cliente, y requerirá en Node.js . Con ES6, ahora podemos utilizar directamente los módulos de todo tipo (AMD) y CommonJS.
