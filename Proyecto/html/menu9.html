Spread Operator

En ES5, podríamos encontrar el máximo de valores en una matriz utilizando el applymétodo de Math.maxla siguiente manera: Math.max.apply(null, [-1, 100, 9001, -32]); // 9001 En ES6, ahora podemos usar el operador se extendió a pasar una matriz de valores que se utilizan como parámetros a una función: Math.max(...[-1, 100, 9001, -32]); // 9001 Podemos concat literales de matriz fácilmente con esta sintaxis intuitiva: let cities = ['San Francisco', 'Los Angeles']; let places = ['Miami', ...cities, 'Chicago']; // ['Miami', 'San Francisco', 'Los Angeles', 'Chicago']

Descripcion

El operador de propagación spread operator permite que una expresión sea expandida en situaciones donde se esperan múltiples argumentos (llamadas a funciones) o múltiples elementos (arrays literales).
