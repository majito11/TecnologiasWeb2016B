var vs let vs const Declaracion de Variables Ejemplo descriptivo

// var es usado en el JavaScript antiguo y tenia algunos problemas. var a = 12; // let es usado con el nuevo Ecmascript 2015 let a = 13; // const es usado para declarar objetos que no se van a modificar, pero si se pueden aumentar const url = 'http://localhost:1337'; // Un arreglo es un ejemplo que se puede aumentar const arrayGrande = ['a','b','c']; arrayGrande.push('d'); // NO hay error const arrayGrande = ['a','b','c','d']; arrayGrande.pop(); // HAY error

Descripcion

Las opciones de var, let y const estan disponibles en el Ecmascript 2015. Cada una tiene algunas peculiaridades.

Peculiaridades

La sentencia let declara una variable de alcance local, la cual, opcionalmente, puede ser inicializada con algún valor.

La palabra reservada let en Mozilla Firefox está solo disponible para bloques de código en HTML envueltos en una etiqueta de script
