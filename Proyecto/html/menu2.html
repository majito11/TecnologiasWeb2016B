ARROW FUNCTIONS

uchas veces tenemos funciones anidadas en las que queremos preservar el contexto de esto desde su ámbito léxico. A continuación se muestra un ejemplo: //function Person(name) { //this.name = name; //} //Person.prototype.prefixName = function (arr) { // return arr.map(function (character) { // return this.name + character; // Cannot read property 'name' of undefined //}); //};// Una solución común a este problema es almacenar el contexto de esto usando una variable: function Person(name) { this.name = name; } Person.prototype.prefixName = function (arr) { var that = this; // Store the context of this return arr.map(function (character) { return that.name + character; }); };

Descripcion

La expresión de función flecha (también conocida como función flecha gruesa) dispone de una sintaxis más corta comparada con la expresión de función convencional y vincula contextualmente el valor de this. Las Funciones Flecha siempre son anónimas.

Peculiaridades

Práctica recomendada: utilice las funciones de flecha siempre que necesite conservar el valor léxico de esto.

Las funciones de flecha también son más concisas cuando se usan en expresiones de función que simplemente devuelven un valor
